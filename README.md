# Менеджер паролей

Простая программа для хранения паролей на компьютере и телефоне.

## Как запустить

1. Скачай все файлы в одну папку
2. Открой файл `index.html` в браузере
3. Всё готово!

## Что умеет

- ✅ Сохранять логины и пароли
- ✅ Показывать и скрывать пароли (глазик 👁️)
- ✅ Генерировать случайные пароли
- ✅ Работать на телефоне и компьютере
- ✅ Устанавливаться как приложение на телефон

## Как пользоваться

### Добавить пароль:
1. Введи сайт (например google.com)
2. Введи логин
3. Введи пароль или нажми "Сгенерировать пароль"
4. Нажми "Сохранить пароль"

### Посмотреть пароли:
- На компьютере: слева список сайтов
- На телефоне: нажми синюю кнопку 📁 внизу справа

### Показать/скрыть пароль:
- Нажми кнопку 👁️ рядом с паролем

### Удалить пароль:
- Нажми красную кнопку "Удалить"

## Важно!

- Все пароли хранятся только в твоём браузере
- Никуда не отправляются в интернет
- Если очистишь историю браузера - пароли удалятся

## Установка на телефон

1. Открой сайт в браузере телефона
2. В меню браузера выбери "Добавить на главный экран"
3. Теперь есть иконка как у обычного приложения

## Если что-то сломалось

- Обнови страницу (F5)
- Проверь, что все файлы в одной папке
- Попробуй в другом браузере

## Файлы

- `index.html` - главная страница
- `style.css` - внешний вид
- `script.js` - программа
- `manifest.json` - настройки для телефона
- `service-worker.js` - для работы без интернета
- `icons/` - картинки для иконок
